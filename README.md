# Rimworld-RationMaking
Adds multiple industrial rations that are cheap alternatives for caravans but has enough caveats that it wouldnt make the Pemmicans and Packaged Survival Meals useless. The Rations are made in the new Ration Maker and unlocked the moment you research Electricity. Inspired by Argon's Salted Meats, Lyn's Hardtacks, and various other MRE mods
